# Plant Disease Dataset

This directory contains sample images of plant diseases organized by plant type and disease. The structure is:

```
disease_dataset/
  ├── tomato/
  │   ├── early_blight/
  │   ├── late_blight/
  │   └── leaf_spot/
  ├── cucumber/
  │   ├── powdery_mildew/
  │   └── downy_mildew/
  ├── potato/
  │   ├── black_scurf/
  │   └── common_scab/
  ├── rice/
  │   ├── blast/
  │   └── bacterial_blight/
  └── wheat/
      ├── rust/
      └── powdery_mildew/
```

Add disease images in JPG format to the corresponding directories. Each image should clearly show the disease symptoms for accurate detection.
